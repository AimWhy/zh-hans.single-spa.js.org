single-spa 生态系统正在迅速发展，以支持尽可能多的框架和构建工具。

## 对框架的支持
越来越多的库可以集成进`用流行框架编写的`应用程序来帮助您做初始化，挂载和卸载。
下面的贡献列表，你可以在自己的项目中自由地用

- [single-spa-angular](./ecosystem-angular.md)

- [single-spa-cycle](./ecosystem-cycle.md)
